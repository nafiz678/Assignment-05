Important FAQ

What is the Document Object Model(DOM)?

Document Object Model is an object model of the whole html.
We can manipulate any element we want through Document Object Model(DOM).

How do you select the element from DOM?

There are many ways to select any element from DOM.
For selecting single element we can use "querySelector()". It only returns the first element that matches the specified selectors.
And the other is "getElementById()". There are 3 ways to select all of the element with the specified name. Those are "getElementsByClassName()", "getElementsByName()", "getElementsByTagName()"

What is the event delegation in the context of DOM, and why is it useful?

Event Delegation is an event-handling pattern that allows us to handle events at higher level in the DOM tree. Its is useful cz we have to use bubbling and some times we have to stop bubbling by Event Delegation.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes by setting "setAttribute()" property. And we can also give style to an element by adding multiple css through "setAttribute()" property.
